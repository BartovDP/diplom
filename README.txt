Инструкция по сборке

- Java SDK 21 версии
    https://www.oracle.com/java/technologies/downloads/?er=221886#jdk21-windows

- Сборщик - Maven
    Версия - 4
    Хотя, работа проверялась только в VS code с расширением Maven for Java
    Ручная установка Maven и запуск через 'mvn clean install' может не сработать

1) Скачать содержимое репозитория

2) Открыть проект в VS code (в качестве директории или в разделе Java projects)

3) В качестве главного класса указать App.java

3) Запустить проект
    ПКМ по Java классу -> 'Run Java'
        или
    ПКМ по проекту в разделе Java projects -> 'Run'
